*MY NOTES*

* HTML :

A) Basics =

1* DOCTYPE: Specifies the type of document (e.g., HTML5) for browser rendering.

2* HTML Tag: Represents the root of the document and contains all the content of the webpage.

3* Headings (h1 to h6): Used to create headings in a document, from largest to smallest in size and importance.

4* Paragraph (p): Defines a block of text as a paragraph.

5* Anchor (a): Creates clickable hyperlinks to navigate between pages or websites.

6* Image (img): Displays images on a webpage.

7* Line Break (br): Inserts a line break in text.

B) Intermediate Elements =

1* Unordered List (ul): Creates a bulleted list of items.

2* Ordered List (ol): Creates a numbered list of items.

3* List Item (li): Represents individual items within a list.

4* Table (table, tr, td, th): Structures tabular data into rows and columns.

5* Forms (form): Collects input from users, such as text, passwords, or selections.

6* Input (input): Accepts user data, such as text, emails, dates, etc.

7* Label (label): Describes the purpose of a form input element.

C) Semantic Tags =

1* Header: Marks the header of a document or section.

2* Nav: Represents a navigation menu with links.

3* Section: Groups content into thematic sections of a document.

4* Article: Represents independent and self-contained content.

5* Footer: Marks the footer of a document or section.

D) Media Elements =

1* Video: Embeds a video on the page with optional controls for playback.

2* Audio: Embeds an audio file on the page, allowing for controls like play and pause.

3* Iframe: Embeds another webpage or external content inside the current page.

E) Advanced Features =

1* Meta Tags: Provide metadata about the webpage (e.g., description, keywords) for SEO and browser configuration.

2* HTML5 Input Types: Specialized inputs for data like emails, colors, dates, and ranges.

3* Script Tag: Embeds or links JavaScript code for interactivity.

4* Style Tag: Embeds CSS code to style the webpage.
